# Target Architecture

Final target:

```text
Human task or issue
 -> Intake
 -> State reader
 -> Horizon reader
 -> Gap analyzer
 -> Planner
 -> Context selector
 -> Codex runner
 -> Patch writer
 -> Git guard
 -> Test runner
 -> Review
 -> Learning
 -> State update
 -> Branch or PR
```

Initial target:

```text
Task string
 -> run folder
 -> static plan
 -> verify
 -> lessons
 -> state update
```
