# Human Preferences

- Build minimal first.
- Use a layered architecture.
- Include a self-learning feedback loop.
- Forge must understand current state and target horizon.
- Forge should use Windows 11, WSL, VS Code, Dev Container, Codex, Codex CLI, and Skills as building blocks.
- GitHub Actions is optional, not central in the first version.
- The build pipeline should be able to build the build pipeline.
