# 05 — Forge CLI Specification

Build a Python command line interface named `forge`.

## Command: `forge status`

Purpose:

Show Forge's self-understanding.

Minimum output:

```text
Forge status
Current maturity: ...
Working capabilities:
- ...
Known limitations:
- ...
Latest run: ...
Next suggested improvement: ...
```

Reads:

```text
.forge/state/current_state.md
.forge/state/capabilities.md
.forge/state/limitations.md
.forge/horizon/maturity_model.md
.forge/backlog/self_improvement_backlog.md
```

## Command: `forge verify`

Purpose:

Run deterministic checks.

Reads:

```text
forge.yml
```

Runs:

```text
runtime.test_command
runtime.lint_command
```

Rules:

- Never run commands generated by Codex.
- Capture output.
- Return non-zero if checks fail.

## Command: `forge run --task "..."`

Purpose:

Create a controlled run.

Minimum behavior:

1. Create run id.
2. Create `.forge/runs/<run-id>/`.
3. Write `task.md`.
4. Snapshot current state.
5. Snapshot horizon.
6. Create simple `plan.md`.
7. Run verification if configured.
8. Write `outcome.md`.

Minimum run folder:

```text
task.md
state_snapshot.md
horizon_snapshot.md
gap_analysis.md
plan.md
test-output.txt
outcome.md
```

Later behavior:

- Call Codex CLI.
- Capture diff.
- Review diff.
- Learn from result.

## Command: `forge improve`

Purpose:

Choose the next self-improvement task.

Inputs:

```text
current state
horizon
maturity model
roadmap
backlog
memory
```

Output:

```text
selected improvement task
reason
acceptance criteria
risk level
```

Optional behavior:

Run the selected task through `forge run`.

## Command: `forge learn --run latest`

Purpose:

Convert run outcome into durable lessons.

Inputs:

```text
run artifacts
test output
review
outcome
```

Writes or appends:

```text
.forge/memory/failure_patterns.md
.forge/memory/success_patterns.md
.forge/memory/coding_rules.md
.forge/backlog/self_improvement_backlog.md
```

## Exit codes

Use predictable exit codes:

```text
0 success
1 general failure
2 config missing or invalid
3 verification failed
4 policy violation
5 run not found
```
