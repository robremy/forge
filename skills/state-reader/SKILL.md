# State Reader Skill

Use this skill before planning any self-improvement.

## Read

```text
.forge/state/current_state.md
.forge/state/capabilities.md
.forge/state/limitations.md
.forge/state/architecture.md
.forge/state/test_health.md
```

## Output

Summarize:

```text
current maturity level
working capabilities
known limitations
latest test health
safe next areas
```

## Rule

Be conservative. Do not claim a capability works unless state and tests support it.
